# Rohan Vitrouthu Portfolio

Three.js portfolio for [rohanvitrouthu.github.io](https://rohanvitrouthu.github.io/), rebuilt with a TRON: Legacy-inspired wireframe grid theme.

## Development

```sh
npm install
npm run dev
```

## Production Build

```sh
npm run build
```

The GitHub Actions workflow in `.github/workflows/deploy.yml` builds the site and deploys the generated `dist` directory to GitHub Pages whenever changes are pushed to `main`.
