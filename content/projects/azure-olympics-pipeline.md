* Engineered an end-to-end cloud data engineering pipeline on Azure Cloud using five major services, including Azure Data Factory, Azure Databricks, and Azure Synapse Analytics, to process and analyze data covering 11,000 athletes.
* Executed data cleansing and transformation logic using Azure Databricks and Apache Spark (PySpark) to ensure data quality—such as correcting data types for medal counts—and staged the final output in Azure Synapse Analytics for subsequent analytical SQL querying and dashboard creation.
