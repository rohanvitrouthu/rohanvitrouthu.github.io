* Engineered a containerized ETL solution using Docker to orchestrate the ingestion of real-time weather data from the OpenMeteo API into a PostgreSQL warehouse.
* Deployed n8n workflows to handle data transformation and scheduling, implementing persistent storage and custom Docker networking for seamless service communication.
* Integrated OAuth2-authenticated alerting via the Gmail API to provide automated notifications based on specific data thresholds.
