I built this blazing-fast, secure portfolio website using the Hugo static site generator.

* **Theme:** PaperMod
* **Features:** Custom Vanilla JS time-based dark/light mode toggle.
* **Hosting:** Deployed via Git CI/CD to GitHub Pages.
